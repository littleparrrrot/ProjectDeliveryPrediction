# ProjectDeliveryPrediction
Исследовательский проект по машинному обучению: проведен EDA датасета, Feature Engineering, заполнены пропуски и выявлены выбросы. В рамках проекта обучили несколько видов моделей для анализа того, как формируются предсказания, на какие признаки модель опирается. Для подбора гиперпараметров использовали optuna, а также lime/shap для анализа фичей
